使用Github和Hexo,Travis CI搭建自动化构建的独立博客站点

发表于 2019-05-29 | 更新于 2019-06-03 | 分类于 实用技巧 | 阅读次数：
本文字数： 5.4k | 阅读时长 ≈ 5 分钟

目标

我们先来看看本文希望达成怎样的目标:

  1. 使用 Hexo 静态页面生成的博客系统搭建个人博客站点。
  2. 使用 Github 的私有仓库作为博客站点的草稿箱，草稿箱内的所有数据不提供公开访问。这样我们可以使用 Github 的 Iuuse 管理工具进行写作的计划安排以及参考资料的收集，同时也可以使用 Pull Request 进行实际的写作管理。
  3. 使用 Github Pages 的公开仓库作为博客站点的对外服务站点。
  4. 整个博客发布流程完全使用免费工具(Azure Devops、Travis CI)全自动化完成。
